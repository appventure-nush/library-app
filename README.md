# Library-App

## Prerequisites

-   Node.js
-   Yarn
-   Preferably VSCode

## Development

# Setup frontend development server

1. Run `cd frontend`
2. Run `yarn install`
3. Run `yarn start`

# Setup backend development server

1. Run `cd backend`
2. Run `npm install`
3. Run `npm start`

## Generators

`yarn run generate container`
`yarn run generate component`
